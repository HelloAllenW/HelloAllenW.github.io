从 Vue 2 开始，template 在运行之前，会被编译成 JavaScript 的 render function。

Vue 推荐在绝大多数情况下使用 template 来创建你的 HTML。然而在一些场景中，就需要使用 render 函数，它比 template 更加灵活。这些 render function 在运行时阶段，就是传说中的 Virtual DOM。
