二月 4, 2017     Web前端
    1.2k Words     5 min

CSS 提供了三种出色的方法可以实现基本的形状。本文就来看看如何使用这些方法来实现一个三角形。三种方法如下：

  • border
  • linear-gradient
  • clip-path
